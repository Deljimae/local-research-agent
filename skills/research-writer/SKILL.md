When using this skill:
- Start with web research and gather only the most relevant sources.
- Prefer current, factual information and cite concrete source URLs.
- Synthesize findings into a concise markdown structure instead of dumping raw notes.
- If the user asks to save the output, use the local write tool after the summary is ready.
- Keep claims traceable to retrieved sources.
